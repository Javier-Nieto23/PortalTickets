# Frontend - Portal de Tickets

Aplicación React para gestión de tickets/servicios.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

## Build para producción

```bash
npm run build
```

## Deploy en Vercel

1. Instala Vercel CLI: `npm i -g vercel`
2. Ejecuta `vercel` en la raíz del proyecto frontend
3. Sigue las instrucciones

O conecta tu repositorio directamente en [vercel.com](https://vercel.com)

### Configuración de Vercel

Asegúrate de configurar la variable de entorno en Vercel:

- `VITE_API_URL`: URL de tu backend en Railway

## Variables de Entorno

Ver `.env.example` para las variables necesarias.
